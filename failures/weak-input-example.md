# Weak Input Example

## Goal
Show how poor or incomplete input can wreck output quality even when the prompt itself is decent.

## Example Weak Input
- Brand: MTS Prompts
- Audience: everyone
- Goal: make it good
- Tone: whatever

## Likely Problems
- generic output
- weak positioning
- vague messaging
- inconsistent tone
- low practical value

## Better Input
- Brand: MTS Prompts
- Audience: small business owners using AI for content and operations
- Goal: explain why structured prompt systems save time and improve consistency
- Tone: confident, practical, and direct

## Takeaway
Garbage in, garbage out still applies. Better prompts help, but better inputs matter too.
