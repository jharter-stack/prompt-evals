# Vague vs Structured Prompt

## Goal
Show the difference between a weak, underspecified prompt and a structured prompt with clear context and output requirements.

## Weak Prompt
Write me some ad copy for my business.

## Problems
- no audience
- no offer
- no tone
- no format
- no platform
- no goal

## Structured Prompt
You are a direct-response copywriter.

Create ad copy for the following:
- Brand: MTS Prompts
- Product: reusable AI prompt systems
- Audience: creators and small business owners
- Offer: starter prompt library
- Tone: confident and modern
- Platform: Instagram

Produce:
1. 5 headline options
2. 3 body copy variations
3. 3 CTA lines

## Expected Difference
The structured version should produce sharper, more usable output because the model has context, direction, and constraints.

## Takeaway
Specificity improves usefulness. Structure improves consistency.
